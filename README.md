# Контрольная работа №4, Сысоев Виталий Вячеславович, ЭФБО-08-24
Описание приложенияp: это простой генератор надёжных паролей, который работает прямо в браузере.   

Возможности:
— Задаёте длину пароля (от 8 до 128 символов);
— Выбираете, какие символы использовать: заглавные и строчные буквы, цифры и спецсимволы;
— Нажимаете кнопку — получаете готовый пароль.   

Особенности:
— Всё работает локально, данные никуда не отправляются;
— Есть проверка: нельзя создать пароль без символов или с недопустимой длиной;
— Написано на чистом HTML, CSS и JavaScript — без лишних библиотек.   

Вывод:приложение получилось простым, удобным и безопасным. Подходит для быстрого создания паролей без установки программ.
